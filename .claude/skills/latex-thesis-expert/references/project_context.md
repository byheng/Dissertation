# 项目背景与核心技术 (Project Context)

## 论文题目
**基于 FPGA 的 SLAM 后端软硬协同加速器设计与实现**

## 核心算法架构 (SchurVINS / SVO 2.0)
*   **框架**：基于 SchurVINS 和 SVO 2.0。
*   **计算模式**：EKF（扩展卡尔曼滤波）位姿更新，采用增量式更新而非全 LM 迭代。
*   **解耦策略**：利用舒尔补（Schur complement）消除路标点，解耦位姿和路标优化。

## 硬件实现 (FPGA 加速器)
*   **Jacobi Engine**：5 级深度流水线（坐标变换 -> 残差计算 -> 雅可比生成 -> 海森矩阵累加）。
*   **SOB 编码 (Sparse Observation Bitmap)**：8 位位图，表示 4 个滑动窗口状态和双目相机的观测情况。
*   **动态门控 (Dynamic Gating)**：利用 SOB 跳过无效计算，包括状态级旁路和相机级时钟门控。
*   **存储优化**：仅存储非零海森块，采用隐式索引节省 BRAM（优化至 ~67 KB）。
*   **Schur 模块**：并行化 $3 \times 3$ 矩阵求逆、中间乘积缓存和对称海森矩阵更新。

## 性能指标
*   **加速比**：相比 ARM CPU 提升约 3.74 倍。
*   **实时性**：后端优化频率 > 25 Hz。
*   **数据集验证**：EuRoC 和 TUM-VI。
